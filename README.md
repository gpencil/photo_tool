# 图片处理工具

一个完全免费、本地运行的图片处理工具，所有处理都在浏览器中完成，保护您的隐私。

## 功能特性

### 1. 图片压缩
- **按质量压缩**：调整图片质量（1-100%）
- **按大小压缩**：压缩到指定大小（KB）

### 2. 尺寸调整
- **按比例缩放**：10%-200% 任意比例缩放
- **固定尺寸**：自定义宽度和高度
- **保持宽高比**：调整时自动保持原始比例

### 3. 美颜和滤镜
- **美白**：调整图片亮度（-50 到 +50）
- **对比度**：增强或减弱对比度（-50 到 +50）
- **饱和度**：调整色彩鲜艳程度（-100 到 +100）
- **锐化**：使图片更清晰（0-100）
- **模糊**：添加模糊效果（0-10）

### 4. 滤镜效果
- 无滤镜
- 黑白
- 复古（Sepia）
- 怀旧（Vintage）
- 暖色调
- 冷色调

### 5. 其他功能
- 实时预览原图和处理后的对比
- 显示图片尺寸和文件大小
- 一键下载处理后的图片
- 重置所有设置

## 快速开始

### 前置要求
- Go 1.24.0 或更高版本

### 安装步骤

1. 启动服务器：
```bash
go run main.go
```

2. 在浏览器中打开：
```
http://localhost:8089
```

### 使用方法

1. **选择图片**：点击"选择图片"按钮，选择要处理的图片
2. **调整参数**：根据需要调整各项参数
   - 压缩：选择按质量或按大小压缩
   - 尺寸：选择按比例或固定尺寸调整
   - 美颜/滤镜：调整各项滑块和选择滤镜
3. **应用处理**：点击"应用处理"按钮
4. **查看效果**：在右侧预览区查看处理后的效果
5. **下载图片**：满意后点击"下载图片"保存

## 技术特点

- **完全本地化**：所有图片处理在浏览器中完成，不上传到任何服务器
- **隐私保护**：您的图片永远不会离开您的设备
- **免费使用**：无需付费，无需注册
- **轻量级**：使用原生 Canvas API，无需依赖第三方库
- **响应式设计**：支持桌面和移动设备

## 注意事项

1. 处理大图片时可能需要较长时间，请耐心等待
2. 建议处理的图片不超过 10MB
3. 锐化和模糊效果对大图片处理较慢
4. 压缩到指定大小功能使用二分查找算法，可能需要几秒钟

## 常见问题

**Q: 图片处理后质量下降？**
A: 可以调整压缩质量参数，使用更高的质量值（如 90-95）。

**Q: 处理后的图片比原图大？**
A: 使用"按大小压缩"功能，设置目标大小。

**Q: 如何保持图片原始比例？**
A: 在"固定尺寸"选项中勾选"保持宽高比"。
