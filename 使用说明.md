# 图片处理工具 - 使用说明

## 启动方法

```bash
go run main.go
```

然后在浏览器访问：**http://localhost:8089**

## 自动处理功能

修改任何参数后，工具会**延迟2秒自动处理**图片，无需手动点击按钮！

### 功能说明

1. **自动处理**
   - 拖动任何滑块后，等待2秒自动处理
   - 修改输入框数值后，等待2秒自动处理
   - 切换单选按钮、下拉选项后，等待2秒自动处理
   - 上传图片后，自动触发第一次处理

2. **立即处理按钮**
   - 如果不想等待2秒，可以点击"立即处理"按钮马上处理
   - 会取消当前的等待计时

3. **处理状态提示**
   - 修改参数时显示：**等待中... (2秒后自动处理)**
   - 处理时显示：**正在处理中，请稍候...**
   - 处理完成后自动隐藏提示

## 操作流程

1. 点击"选择图片"上传照片
2. 图片自动加载并处理
3. 调整任何参数（压缩、尺寸、美颜、滤镜等）
4. 等待2秒，或点击"立即处理"
5. 查看右侧处理效果
6. 点击"下载图片"保存结果

## 优势

- 🚀 **即时反馈**：不用反复点击处理按钮
- ⏱️ **防抖设计**：快速调整参数时不会频繁处理
- 👀 **状态可视**：清楚知道当前处理状态
- 🎨 **流畅体验**：类似专业图片编辑软件

## 延迟时间

默认延迟 **2秒**，这是一个平衡点：
- 不会太快（避免拖动滑块时频繁处理）
- 不会太慢（不用等太久看到效果）

如果需要修改延迟时间，编辑 `static/app.js` 第40行：
```javascript
}, 2000);  // 改成其他值，单位是毫秒（1000毫秒 = 1秒）
```

## 注意事项

- 大图片处理需要时间，请耐心等待
- 处理过程中可以继续调整参数，会重新计时
- 锐化和模糊效果对大图片处理较慢
